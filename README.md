## Установка
1.Скачайте игру из релизов https://github.com/Kirillovo-Company/SandVox/releases
2.Распакуйте архив в любое место
3.Скачайте Python если еще нету(установщик в папке)
4.Запустите bat файл

## Как сообщить о Баге
